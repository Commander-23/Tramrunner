# Departures JSON Format

The response for departures dict of the queried stop. The Response has the following keys

```json
{
    "Name": "Hauptbahnhof",
    "Status": {"Code": "Ok"},
    "Place": "Dresden",
    "ExpirationTime": "/Date(1754014402764+0200)/",
    "Departures": [{}, {}, {}]
}
```
